又一个知乎日报 iOS 客户端。

![Main Page 2](Screenshots/MainPage2.png)

## 项目特色

- 使用 `Objective-C`
- 纯代码实现，无 `Storyboard` 和 `nib` 文件
- 没有任何第三方库的依赖
- 尽可能的实现 MVC 设计模式
- 界面设计仿照官方应用，但尽可能保持简单
- 本项目会持续更新
